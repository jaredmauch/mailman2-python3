Этот текст может включать строки подстановки Python (текст по-английски), которые заменяются следующими атрибутами списка рассылки:
  • real_name — "красивое" имя списка рассылки; обычно это имя списка рассылки с заглавной буквы.
  • list_name — имя, которое используется в адресах страниц, регистр символов имеет значение.
  • host_name — полное имя домена (FQDN) списка рассылки.
  • web_page_url — базовый адрес для Mailman. Именно к этому адресу нужно добавлять, например, listinfo/%(list_name)s чтобы получить адрес информационной страницы списка рассылки.
  • description — краткое описание списка рассылки.
  • info — полное описание списка рассылки.
  • cgiext — расширение файлов CGI-сценариев.
